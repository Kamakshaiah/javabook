Rules for Naming Variables in Java

Java programming language has its own set of rules and conventions for naming variables. Here's what you need to know:

  1. Variables in Java are case-sensitive.
  2. A variable's name is a sequence of Unicode letters and digits. It can begin with a letter, $ or _. However, it's a convention to begin a variable name with a letter. Also, variable names cannot use whitespace in Java.
  3. When creating variables, choose a name that makes sense. For example, score, number, level makes more sense than variable names such as s, n, and l.
  4. If you choose one-word variable names, use all lowercase letters. For example, it's better to use speed rather than SPEED, or sPEED.
  5. If you choose variable names having more than one word, use all lowercase letters for the first word and capitalize the first letter of each subsequent word. For example, speedLimit.

There are 4 types of variables in Java programming language:

  1. Instance Variables (Non-Static Fields)
  2. Class Variables (Static Fields)
  3. Local Variables
  4. Parameters
