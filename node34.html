Access Modifiers

As the name suggests access modifiers in Java helps to restrict the scope of a class, constructor , variable , method or data member. There are four types of access modifiers available in java.

  1. Default (No keyword required)
  2. Private
  3. Protected
  4. Public


Table 2.1: Access Modifiers and their properties
private default protected public
Same class yes yes yes yes
Same pkg. sub. class no yes yes yes
Same pkg. non sub. class no yes yes yes
Diff. pkg. sub. class no no yes yes
Diff. pkg. non sub. class no no no yes
